Step 1 — Brand Info: Enter the brand name, select the platform (Shopify, WooCommerce, CSV, or Custom), and the store URL.

Step 2 — Credentials: Enter the API credentials for the platform. For Shopify, you'll need an Access Token. For WooCommerce, a Consumer Key and Secret.

Step 3 — Alerts: Optionally add a Slack webhook URL, alert email, and set the default alert threshold (default: 7 days of stock remaining).

💡 Don't have API keys yet? Choose "CSV Upload Only" as the platform. You can upload inventory data manually via CSV and connect the API later.
